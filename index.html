Формат ссылки для бота

Бот берёт subscription_key из Supabase и собирает ссылку на сайт с параметром ?to=. В примере ниже использован твой ключ. Клиент сначала видит сайт, но при копировании получает ссылку, которую можно использовать для HAPP.

Что выдаёт бот:

https://vpn-sitess.vercel.app/?to=happ%3A%2F%2Fadd%2Fhttps%3A%2F%2Fvpn-sitess.vercel.app%2Fapi%2Fsub%2FFBxXE6TZ2oGq9ge3f2tsfg

Что копируется для Happ:

happ://add/https://vpn-sitess.vercel.app/api/sub/FBxXE6TZ2oGq9ge3f2tsfg
